CocoaBench

A Robust Framework for Evaluating Agentic Reasoning

Paper GitHub Dataset

We introduce CocoaBench, a new benchmark designed to test the limits of Large Language Models in complex, multi-step reasoning tasks. Unlike previous benchmarks that focus on single-turn QA, CocoaBench simulates real-world software engineering environments.

Why CocoaBench?

Current evaluations often saturate quickly. CocoaBench introduces dynamic environment updates, requiring agents to not only generate code but also perceive changes in the state.
